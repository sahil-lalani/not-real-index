# Not Real Index

This is the Not Real Index. Thanks to Alpha Beta Sigma for sponsoring.

To run locally:

1. Clone the repo

```bash
git clone https://github.com/sahil-lalani/not-real-index.git
```

2. Install dependencies

```bash
npm install
```

```bash
npm run dev
```

To contribute:

1. Fork the repo
2. Submit a PR
